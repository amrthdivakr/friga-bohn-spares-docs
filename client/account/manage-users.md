# How to Manage User in the Website
#### [Made by Amruth Divakar with Scribe](https://scribehow.com/o/AmjRagUGQxOh31NKNgqRAQ/viewer/How_to_Manage_User_in_the_Website__ti6LMcV2QQmY_ykFEp4BMg)
Follow this guide to learn the necessary steps for adding/update users to your Shopify account. You will find instructions on assigning specific roles and locations to ensure your team has the correct system access.

1\. Navigate to [[Account]] Page

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/c7751c63-3790-4b37-965e-624719fce213/ascreenshot_bc30cc476841409381c16833f3335355_text_export.jpeg)


2\. Click "Users"

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/c7751c63-3790-4b37-965e-624719fce213/ascreenshot_ef006fca74484b06be4538a84f7d2851_text_export.jpeg)


3\. Click "Add User" to add a new User

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/fd66d51b-4cd0-4245-a632-da13c62a4eb9/ascreenshot_349a40730d664a7c8a5e9c63cafdf81a_text_export.jpeg)


4\. Add User details

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/b872b809-5b74-431e-862b-31cc29df309a/ascreenshot_c87aca68565c4fa7ac20da372a94c10a_text_export.jpeg)


5\. Click "Select a role"

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/0d6536f5-01b3-4caf-8d75-7c7dc61589fb/ascreenshot_2e58eae998e544808dc79fc151b0412d_text_export.jpeg)


6\. Select a role

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/e0f3d4ab-5d9d-4ba6-84de-074bc9f037a2/ascreenshot_36b361efb1da45949b1b5bddad0347c0_text_export.jpeg)


7\. Select a Company (access only available for Accounts and Web Admins, defaults to own company for Distributor Admin )

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/3b3a4e24-6117-4009-9bec-fd6679e6524b/ascreenshot_cabdead4ecae4ea08fbc1673d0f64cb4_text_export.jpeg)


8\. Click "Select a location"

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/0607cd6d-54f7-405a-80e9-26e94b855c5b/ascreenshot_a4d0d776c5614818a2a7964a5fa039e5_text_export.jpeg)


9\. Select a company location for the user.

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/a1e7b0dd-1b54-4902-9c78-3d12dbb7c886/ascreenshot_9f89cfd3b1814b46ae18fed8e050096c_text_export.jpeg)


10\. Click "Create"

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/da7f119f-56d2-423f-aaf8-77174d57cccc/ascreenshot_8fcd5c7efb164f4fa1208554ee984b7b_text_export.jpeg)


11\. Click this icon to edit a User.

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-22/55d03eb3-a68b-4995-9347-9432ed77e10d/ascreenshot_79f3f94e009c4fec9a26521484dac665_text_export.jpeg)
