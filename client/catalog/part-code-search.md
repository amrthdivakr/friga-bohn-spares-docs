# How To Search For Parts Using Part Numbers
#### [Made by Amruth Divakar with Scribe](https://scribehow.com/o/AmjRagUGQxOh31NKNgqRAQ/viewer/How_To_Search_For_Parts_Using_Part_Numbers__aPxIfoHdSK-hFiceij7W-g)
Learn how to efficiently locate specific components on your Shopify store using the dedicated search field. This guide walks you through the quick process of entering a part number to find exactly what you need in seconds.

1\. Navigate to [Parts](https://staging-28eafe2bb41e547cf237.o2.myshopify.dev/parts) page

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-14/05554cac-b86a-4a44-ba12-e99e6009e404/ascreenshot_bd9ceeef138e4d338b4ab5cb26ec64b2_text_export.jpeg)


2\. Click the [[Search Box]] field.

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-14/05554cac-b86a-4a44-ba12-e99e6009e404/ascreenshot_91bf09f893bf46449b12051e6360c19f_text_export.jpeg)


3\. Type Part number like, "pdve00794" followed by  [[Enter]] or clicking the [[Search ]]button


4\. Search will return the part with the entered part number

![](https://colony-recorder.s3.amazonaws.com/files/2026-06-14/7e85c83b-2e6f-43c4-83e1-68eca136569b/ascreenshot_011567e996a64ebc8227c5942ea3cb76_text_export.jpeg)
