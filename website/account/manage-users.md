# How to Manage User in the Website

Follow this guide to learn the necessary steps for adding/update users to your Shopify account. You will find instructions on assigning specific roles and locations to ensure your team has the correct system access.

1\. Navigate to **Account** Page

![](../images/account/manage-users/step-01.jpeg)


2\. Click **Users**

![](../images/account/manage-users/step-02.jpeg)


3\. Click **Add User** to add a new User

![](../images/account/manage-users/step-03.jpeg)


4\. Add User details

![](../images/account/manage-users/step-04.jpeg)


5\. Click **Select a role**

![](../images/account/manage-users/step-05.jpeg)


6\. Select a role

![](../images/account/manage-users/step-06.jpeg)


7\. Select a Company (access only available for Accounts and Web Admins, defaults to own company for Distributor Admin )

![](../images/account/manage-users/step-07.jpeg)


8\. Click **Select a location**

![](../images/account/manage-users/step-08.jpeg)


9\. Select a company location for the user.

![](../images/account/manage-users/step-09.jpeg)


10\. Click **Create**

![](../images/account/manage-users/step-10.jpeg)


11\. Click the edit icon to edit a User.

![](../images/account/manage-users/step-11.jpeg)
